# Home App

## Overview

The `home` app manages the public-facing pages of the CattleCare platform.

It includes:

- Homepage
- About page
- Contact page
- Floating AI chatbot
- Landing UI components

---

# Features

## Homepage

Modern landing page with:

- Hero section
- Live monitoring preview
- Statistics section
- Features showcase
- CTA sections

---

## About Page

Includes:

- Mission statement
- Team members
- Company information
- Feature highlights

---

## Contact Page

- Contact form
- Email support
- Phone support
- Address details

---

## Floating AI Chatbot

Features:

- AI assistant UI
- Real-time responses

- Device diagnostics help
- Smart farming guidance
